สวัสดีครับ เว็บไซต์นี้เป็นส่วนหนึ่งของโปรเจกต์จบคอร์ส Fundamental Web Dev จาก BorntoDevครับ

สิ่งที่ได้จากการเรียนรู้คอร์สนี้

  • สามารถนำความรู้ด้านการเขียนเว็บไซต์ไปต่อยอดได้
  • ได้ความรู้เกี่ยวกับภาษา HTML CSS เบื้องต้น
  • ได้ทักษะ ความสามารถเพิ่มขึ้น
  • สามารถพัฒนาตัวเองต่อไปในด้าน Dev ได้
